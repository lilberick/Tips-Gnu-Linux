# [JAAMSIM](https://jaamsim.com/)

## Archivo para simulación con JAAMSIM: CarguioSimple.cfg

Es necesario que estos 3 archivos estén ubicados en la misma carpeta

```sh
CarguioSimple
|__ camion.png
|__ CarguioSimple.cfg
|__ pala.png
```
